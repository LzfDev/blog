# blog
Python3 基于 Flask 框架搭建个人博客

# 系统简介
使用 Python 语言及 Flask 框架开发一个简单的博客系统。涉及 Flask Web 开发，使用 Peewee 构建数据模型，简单的 Jinja2 模板， Pygments 模块实现代码语法高亮，MarkDown 格式编写博客， Micawber 实现多媒体播放。

# 功能描述
实现一个具备基本功能的博客系统。支持：
- 创建，编辑及删除博文
- 博文编辑支持Markdown格式
- 博文显示支持语法高亮
- 支持全文搜索
- 支持分页显示
- 支持保存草稿
- 支持登录和登出
